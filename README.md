WebGL from scratch
===

### Dev environment

 * [MongoDB](https://www.mongodb.com/download-center/community) >= 4.0
 * [Node.js](https://nodejs.org/en/download/) >= 10.15
 * [Yarn](https://yarnpkg.com/en/docs/install) >= 1.15

```bash
git clone https://github.com/danielesteban/WebGL.git
cd WebGL
yarn install
npm start
```
